The Argent Vault - Site Plan

This document outlines the planning for The Argent Vault website project.

Site Name

The Argent Vault

The name "The Argent Vault" was selected because:

  • "Argent" is derived from Latin for silver, representing value and financial resources
  • "Vault" suggests security, storage, and protection of assets
  • Together, they create a memorable brand that conveys financial security and knowledge preservation

Optional domain availability: theargentvault.com or argentvault.org

Site Purpose

The Argent Vault will provide clear, accessible financial education for young adults and college students. The site will:

  • Offer fundamental personal finance information in an easy-to-understand format
  • Provide interactive tools like a budget calculator to help users apply financial concepts
  • Display live financial data through API integration to connect learning with real-world markets
  • Serve as a centralized resource for financial literacy, covering budgeting, investing, and money management basics

Scenarios

These scenarios represent questions from our target audience of young adults and college students:

  1. "I just got my first paycheck from my part-time job. How much should I save versus spend?"
  2. "I've heard I should start investing early, but I don't know anything about stocks or how to begin. Where should I start?"
  3. "What's the difference between a need and a want when creating a budget?"
  4. "How can I track my spending to make sure I'm staying within my budget?"

Color Scheme

The selected color scheme for The Argent Vault website:

Primary
#2c3e50

Primary Color (#2c3e50 - Dark Blue): Used for headings, navigation elements, and footer background.

Secondary
#3498db

Secondary Color (#3498db - Bright Blue): Used for links, buttons, and interactive elements.

Accent
#f1c40f

Accent Color (#f1c40f - Gold): Used for call-to-action buttons, highlights, and important notices.

Background
#ecf0f1

Background Color (#ecf0f1 - Light Gray): Used for page background and content section backgrounds.

Text
#2c3c3d

Text Color (#2c3c3d - Dark Gray): Used for body text and general content.

Typography

The selected fonts for The Argent Vault website:

Georgia (Serif Font)

Usage: All headings (H1, H2, H3, etc.) to create a professional, trustworthy appearance.

Segoe UI (Sans-serif Font)

Usage: Body text, navigation, buttons, and all other text elements for optimal readability.

Note: This site plan document uses the exact typography specified above.

Files to Create

To implement The Argent Vault website, the following files will be created:

  • index.html - Home page with introduction and navigation
  • budgeting.html - Budgeting Basics page with interactive calculator
  • investing.html - Investing 101 page with live market data
  • styles/ - Directory for CSS files
    • main.css - Primary stylesheet for all pages
    • calculator.css - Specific styles for the budget calculator
    • responsive.css - Media queries for responsive design
  • scripts/ - Directory for JavaScript files
    • main.js - Common functionality across all pages
    • calculator.js - Budget calculator logic
    • api.js - Financial API integration for market data
  • images/ - Directory for all website images and graphics
